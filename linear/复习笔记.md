## 一、线性回归基础

### 1.1 核心概念
- **线性回归**：监督学习算法，用于预测连续值标签
- **一元线性回归**：`h = wx + b`
- **多元线性回归**：`h = w₁x₁ + w₂x₂ + ... + wₙxₙ + b`
- **w**：权重系数，**b**：偏置项

### 1.2 关键公式
| 名称 | 公式 | 说明 |
|------|------|------|
| 预测值 | $h = wx + b$ | 线性组合 |
| 均方误差 MSE | $\frac{1}{n}\sum(y_{pred} - y_{true})^2$ | 回归常用损失函数 |
| 均方根误差 RMSE | $\sqrt{MSE}$ | 与标签同单位 |
| 平均绝对误差 MAE | $\frac{1}{n}\sum|y_{pred} - y_{true}|$ | 对异常值更鲁棒 |

### 1.3 sklearn API
```python
from sklearn.linear_model import LinearRegression

estimator = LinearRegression()
estimator.fit(x_train, y_train)      # 训练
estimator.predict(x_test)            # 预测
estimator.coef_                     # 权重 w
estimator.intercept_                # 偏置 b
```

## 二、梯度下降

### 2.1 核心概念
- **梯度**：函数在某点增长最快的方向
- **学习率(η)**：控制每步更新幅度
  - 过大 → 震荡不收敛
  - 太小 → 收敛慢

### 2.2 梯度下降分类
| 方法 | 特点 | 优点 | 缺点 |
|------|------|------|------|
| FGD 全梯度下降 | 每次用全部数据 | 稳定向最优 | 计算量大 |
| SGD 随机梯度下降 | 每次用1个样本 | 快、大数据友好 | 震荡不稳定 |
| MBGD 小批量 | 每次用 batch 个样本 | 平衡速度和稳定 | 需调 batch size |
| SAG | 存储历史梯度平均 | 稳定 | 内存消耗大 |

### 2.3 sklearn SGD API
```python
from sklearn.linear_model import SGDRegressor

estimator = SGDRegressor(max_iter=1000, tol=1e-3,
                          learning_rate='constant', eta0=0.01)
```

## 三、拟合问题

### 3.1 三种状态
| 状态 | 原因 | 表现 |
|------|------|------|
| **欠拟合** | 模型太简单 | 训练/测试误差都高 |
| **恰好拟合** | 模型复杂度合适 | 泛化能力强 |
| **过拟合** | 模型太复杂/特征过多 | 训练误差低，测试误差高 |

### 3.2 欠拟合 → 增加特征
```python
# 原来：X = x.reshape(-1, 1)
# 增加高次项特征
X2 = np.hstack([X, X**2])           # 二次多项式
X3 = np.hstack([X, X**2, X**3, ...])  # 高次多项式
```

## 四、正则化（解决过拟合）

### 4.1 L1 正则化 (Lasso)
- **公式**：$Loss = MSE + \alpha \sum|w_i|$
- **特点**：使部分权重 **为0**，实现**特征选择**
- **适用**：特征多，需要筛选重要特征

### 4.2 L2 正则化 (Ridge 岭回归)
- **公式**：$Loss = MSE + \alpha \sum w_i^2$
- **特点**：让权重**尽可能小但不为0**
- **适用**：多重共线性问题

### 4.3 sklearn API
```python
from sklearn.linear_model import Lasso, Ridge

# L1 正则化
lasso = Lasso(alpha=0.2)    # alpha 越大，正则化越强

# L2 正则化
ridge = Ridge(alpha=100)
```

### 4.4 正则化强度选择
- `alpha` 越大 → 正则化越强 → 权重越小
- 太小可能仍过拟合，太大会欠拟合
