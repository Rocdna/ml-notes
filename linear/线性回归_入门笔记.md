# 线性回归入门笔记

## 1. 什么是线性回归？

线性回归是一种**监督学习**算法，用于预测**连续型**数值输出。

**核心思想**：找到一条直线（或一个平面/超平面），使得预测值与真实值之间的误差最小。

**应用场景**：
- 房价预测
- 销售额预测
- 温度预测
- 任何连续值预测

---

## 2. 线性回归的数学形式

### 2.1 一元线性回归（单特征）

$$y = wx + b$$

- $y$：目标变量（预测值）
- $x$：特征变量（输入）
- $w$：权重（斜率）
- $b$：偏置（截距）

### 2.2 多元线性回归（多特征）

$$y = w_1x_1 + w_2x_2 + ... + w_nx_n + b$$

用向量表示：

$$y = \mathbf{w}^T \mathbf{x} + b$$

---

## 3. 损失函数

线性回归使用**均方误差（MSE）**作为损失函数：

$$MSE = \frac{1}{m} \sum_{i=1}^{m} (y_i - \hat{y}_i)^2$$

其中 $\hat{y}_i = wx_i + b$

**为什么用MSE？**
- 计算方便，求导简单
- 对大误差惩罚更重
- 基于正态分布假设的最优估计

---

## 4. 如何求解？两种主要方法

### 4.1 最小二乘法（Normal Equation）

直接求解最优解：

$$\mathbf{w} = (\mathbf{X}^T \mathbf{X})^{-1} \mathbf{X}^T \mathbf{y}$$

**优点**：
- 直接求出解析解
- 不需要迭代

**缺点**：
- 计算复杂度高 $O(n^3)$
- 当特征数量大时很慢
- 可能出现矩阵不可逆

### 4.2 梯度下降法（Gradient Descent）

迭代更新参数：

$$w = w - \alpha \cdot \frac{\partial}{\partial w}MSE$$

$$\alpha = \text{学习率}$$

**优点**：
- 适合大规模数据
- 可以处理非线性（通过特征工程）

**缺点**：
- 需要选择学习率
- 可能陷入局部最优（线性回归是凸函数，没问题）

---

## 5. 模型评估指标

### 5.1 MSE（均方误差）

$$MSE = \frac{1}{m} \sum (y_i - \hat{y}_i)^2$$

### 5.2 RMSE（均方根误差）

$$RMSE = \sqrt{MSE}$$

### 5.3 MAE（平均绝对误差）

$$MAE = \frac{1}{m} \sum |y_i - \hat{y}_i|$$

### 5.4 R²（决定系数）

$$R^2 = 1 - \frac{\sum(y_i - \hat{y}_i)^2}{\sum(y_i - \bar{y})^2}$$

- R² = 1：完美预测
- R² = 0：只用均值预测
- R² < 0：比均值预测还差

## 6. Python实现

### 使用NumPy手动实现

```python
import numpy as np

class LinearRegression:
    def __init__(self):
        self.w = None
        self.b = None

    def fit(self, X, y):
        m = len(X)
        # 最小二乘法
        w = np.sum((X - np.mean(X)) * (y - np.mean(y))) / np.sum((X - np.mean(X)) ** 2)
        b = np.mean(y) - w * np.mean(X)
        self.w = w
        self.b = b
        return self

    def predict(self, X):
        return self.w * X + self.b

# 使用
X = np.array([1, 2, 3, 4, 5])
y = np.array([2, 4, 5, 4, 5])
model = LinearRegression().fit(X, y)
print(model.w, model.b)  # 斜率和截距
print(model.predict(6))  # 预测新值
```

### 使用sklearn

```python
from sklearn.linear_model import LinearRegression
from sklearn.model_selection import train_test_split
from sklearn.metrics import mean_squared_error, r2_score

# 划分数据
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2)

# 训练模型
model = LinearRegression()
model.fit(X_train, y_train)

# 预测
y_pred = model.predict(X_test)

# 评估
print(f"R²: {r2_score(y_test, y_pred)}")
print(f"MSE: {mean_squared_error(y_test, y_pred)}")
```

## 7. 重要注意事项

### 7.1 特征缩放
- 梯度下降时，特征尺度差异大会导致收敛慢
- 使用**标准化**或**归一化**

### 7.2 过拟合
- 特征太多，模型复杂度过高
- 解决方案：正则化（Ridge、Lasso）

### 7.3 线性假设
- 线性回归假设特征与目标是线性关系
- 非线性关系需要**特征工程**（多项式、交互特征）
